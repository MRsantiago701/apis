# Proyecto de APIs

Este repositorio contiene diferentes ejemplos de consumo y desarrollo de APIs utilizando Node.js y Express.

## API de Mapa
Permite visualizar la ubicación de la Universidad mediante un mapa interactivo, facilitando la identificación del plantel y su entorno.

## API de Redes Sociales
Muestra publicaciones realizadas por distintos usuarios, simulando el funcionamiento básico de una plataforma de red social.

## API de Base de Datos
Permite consultar información almacenada en una base de datos y visualizar sus tablas y registros de manera organizada.

## API de Streaming
Presenta un catálogo de contenido multimedia, simulando una plataforma de streaming donde los usuarios pueden explorar diferentes opciones disponibles.

## Tecnologías Utilizadas
- Node.js
- Express.js
- JavaScript
